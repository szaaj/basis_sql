SQL Cheat Sheet

Alle SQL commando's op één plek

Snelle Navigatie

SELECT WHERE Operators AND/OR/NOT ORDER BY LIMIT INSERT UPDATE DELETE Functies JOINs GROUP BY

📊 SELECT - Data Ophalen

Alle kolommen
SELECT * FROM tabel;
Specifieke kolommen
SELECT kolom1, kolom2, kolom3 FROM tabel;
DISTINCT - Unieke waarden
SELECT DISTINCT plaats FROM klant;
AS - Alias (hernoemen)
SELECT naam AS klantnaam, plaats AS woonplaats FROM klant;

🔍 WHERE - Filteren

Basis filter
SELECT * FROM klant WHERE plaats = 'Amsterdam';

⚖️ Vergelijkingsoperatoren

Operator Betekenis Voorbeeld
= Gelijk aan WHERE prijs = 500
<> of != Niet gelijk aan WHERE plaats <> 'Utrecht'
> Groter dan WHERE prijs > 1000
< Kleiner dan WHERE prijs < 500
>= Groter of gelijk WHERE dagen >= 7
<= Kleiner of gelijk WHERE prijs <= 2000
BETWEEN - Tussen twee waarden
SELECT * FROM reis WHERE prijsperpersoon BETWEEN 500 AND 1500;
IN - Een van meerdere waarden
SELECT * FROM klant WHERE plaats IN ('Amsterdam', 'Rotterdam', 'Utrecht');
LIKE - Patroon matching
SELECT * FROM klant WHERE naam LIKE 'Jan%';  -- % = wildcard
IS NULL / IS NOT NULL
SELECT * FROM klant WHERE email IS NULL;
SELECT * FROM klant WHERE email IS NOT NULL;

🔗 AND, OR, NOT - Combineren

AND - Beide waar
SELECT * FROM reis 
WHERE aantaldagen = 7 AND prijsperpersoon < 1000;
OR - Een van beide waar
SELECT * FROM klant 
WHERE plaats = 'Amsterdam' OR plaats = 'Rotterdam';
NOT - Omkeren
SELECT * FROM bestemming WHERE NOT land = 'Nederland';
Haakjes voor complexe filters
SELECT * FROM reis 
WHERE (werelddeel = 'Europa' OR werelddeel = 'Azië') 
AND prijsperpersoon < 1000;

🔄 ORDER BY - Sorteren

Oplopend (A-Z, 0-9) - ASC
SELECT * FROM klant ORDER BY naam ASC;
Aflopend (Z-A, 9-0) - DESC
SELECT * FROM reis ORDER BY prijsperpersoon DESC;
Meerdere kolommen
SELECT * FROM klant ORDER BY plaats ASC, naam ASC;

🎯 LIMIT - Aantal Beperken

Top 10
SELECT * FROM reis ORDER BY prijsperpersoon ASC LIMIT 10;

➕ INSERT - Data Toevoegen

Enkele rij
INSERT INTO klant (idklant, naam, plaats)
VALUES (100, 'Jansen P.', 'Amsterdam');
Meerdere rijen
INSERT INTO klant (idklant, naam, plaats)
VALUES 
    (101, 'Bakker A.', 'Rotterdam'),
    (102, 'Visser B.', 'Utrecht'),
    (103, 'Smit C.', 'Den Haag');

✏️ UPDATE - Data Wijzigen

⚠️ ALTIJD een WHERE gebruiken! Anders wijzig je ALLE rijen!

Enkele kolom
UPDATE klant 
SET telefoon = '0612345678' 
WHERE idklant = 5;
Meerdere kolommen
UPDATE klant 
SET naam = 'Jansen-Bakker', email = 'jb@email.nl' 
WHERE idklant = 5;
Met berekening
UPDATE reis 
SET prijsperpersoon = prijsperpersoon * 1.10 
WHERE idreis = 5;

🗑️ DELETE - Data Verwijderen

⚠️⚠️⚠️ ALTIJD een WHERE gebruiken! Anders verwijder je ALLE rijen!

Specifieke rij
DELETE FROM klant WHERE idklant = 5;
Met voorwaarde
DELETE FROM reis WHERE prijsperpersoon > 5000;

🧮 Aggregatiefuncties

Functie Wat doet het? Voorbeeld
COUNT() Telt aantal rijen SELECT COUNT(*) FROM klant
SUM() Totaal optellen SELECT SUM(prijs) FROM reis
AVG() Gemiddelde SELECT AVG(prijs) FROM reis
MIN() Kleinste waarde SELECT MIN(prijs) FROM reis
MAX() Grootste waarde SELECT MAX(prijs) FROM reis

🔗 JOINs - Tabellen Combineren (Periode 2)

INNER JOIN - Overeenkomende rijen
SELECT r.*, b.plaats, b.land
FROM reis r
INNER JOIN bestemming b ON r.idbestemming = b.idbestemming;

📦 GROUP BY - Groeperen (Periode 2)

Groeperen met COUNT
SELECT plaats, COUNT(*) as aantal
FROM klant
GROUP BY plaats;
HAVING - Filter op groepen
SELECT plaats, COUNT(*) as aantal
FROM klant
GROUP BY plaats
HAVING COUNT(*) > 5;

📐 Volgorde van Keywords

  1. SELECT - Welke kolommen
  2. FROM - Uit welke tabel
  3. JOIN - Tabellen combineren (optioneel)
  4. WHERE - Rijen filteren
  5. GROUP BY - Groeperen (optioneel)
  6. HAVING - Groepen filteren (optioneel)
  7. ORDER BY - Sorteren
  8. LIMIT - Aantal beperken

🐛 Veelgemaakte Fouten

❌ Quotes vergeten bij tekst

WHERE plaats = Amsterdam
WHERE plaats = 'Amsterdam'

❌ WHERE na ORDER BY

SELECT * FROM reis ORDER BY prijs WHERE dagen = 7
SELECT * FROM reis WHERE dagen = 7 ORDER BY prijs

❌ UPDATE/DELETE zonder WHERE

UPDATE klant SET plaats = 'Amsterdam'
UPDATE klant SET plaats = 'Amsterdam' WHERE idklant = 5
