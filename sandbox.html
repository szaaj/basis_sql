🧪 Wat is de Sandbox?

Dit is een vrije omgeving waar je zonder beperkingen kunt experimenteren met SQL queries. Er zijn geen oefeningen, geen validatie - gewoon proberen!

📋 Handige Query Voorbeelden

Eerste 10 klanten
SELECT * FROM klant LIMIT 10;
Aantal klanten tellen
SELECT COUNT(*) as totaal FROM klant;
Klanten per stad
SELECT plaats, COUNT(*) as aantal
FROM klant
GROUP BY plaats
ORDER BY aantal DESC;
Reizen tussen €500-€1500
SELECT * FROM reis
WHERE prijsperpersoon BETWEEN 500 AND 1500
ORDER BY prijsperpersoon ASC;
