05

JOINs - Tabellen Combineren

Gebruik INNER, LEFT en RIGHT JOIN

Voortgang 0/5 oefeningen

💡 Wat ga je leren?

Tot nu toe heb je met één tabel gewerkt. Maar wat als je data uit meerdere tabellen wilt combineren? Daarvoor gebruik je JOINs!

Database Structuur

Onze bioscoop database heeft 4 tabellen:

  • film - films met filmid, filmnaam, genreid, regisseur, duur
  • genre - genres met genreid, genrenaam
  • klant - klanten met klantid, voornaam, achternaam
  • reservering - reserveringen met reserveringid, klantid, filmid, datum

INNER JOIN - Alleen Matches

INNER JOIN toont alleen rijen waar een match is in BEIDE tabellen:

Films met genre naam 
SELECT film.filmnaam, genre.genrenaam
FROM film
INNER JOIN genre ON film.genreid = genre.genreid;

LEFT JOIN - Alles van Links

LEFT JOIN toont ALLE rijen van de linker tabel, ook zonder match:

Alle klanten + hun reserveringen (ook klanten zonder) 
SELECT klant.voornaam, COUNT(reservering.reserveringid) AS aantal
FROM klant
LEFT JOIN reservering ON klant.klantid = reservering.klantid
GROUP BY klant.klantid, klant.voornaam;

RIGHT JOIN - Alles van Rechts

RIGHT JOIN toont ALLE rijen van de rechter tabel:

Werkt hetzelfde maar gespiegeld 
SELECT ...
FROM tabel1
RIGHT JOIN tabel2 ON ...
💡 JOIN Syntax: 
SELECT kolommen
FROM linker_tabel
[INNER/LEFT/RIGHT] JOIN rechter_tabel 
    ON linker.key = rechter.key;

🎯 Oefeningen

Oefening 1 - Films met genre naam

Gemiddeld

Combineer de film en genre tabel. Toon filmnaam en genrenaam. Gebruik INNER JOIN.

SQL Query
💡 JOIN ON film.genreid = genre.genreid
💡 Hint

INNER JOIN genre ON film.genreid = genre.genreid

✅ Oplossing 
SELECT film.filmnaam, genre.genrenaam
FROM film
INNER JOIN genre ON film.genreid = genre.genreid;

Oefening 2 - Klanten met hun reserveringen

Gemiddeld

Combineer klant en reservering. Toon voornaam, achternaam en datum. Gebruik INNER JOIN (alleen klanten MET reservering).

SQL Query
💡 JOIN ON klantid
💡 Hint

INNER JOIN reservering ON klant.klantid = reservering.klantid

✅ Oplossing 
SELECT klant.voornaam, klant.achternaam, reservering.datum
FROM klant
INNER JOIN reservering ON klant.klantid = reservering.klantid;

Oefening 3 - Aantal reserveringen per klant

Moeilijk

Tel hoeveel reserveringen elke klant heeft. Gebruik LEFT JOIN (ook klanten zonder reservering). Toon voornaam, achternaam en aantal_reserveringen.

SQL Query
💡 LEFT JOIN + GROUP BY + COUNT
💡 Hint

LEFT JOIN, dan GROUP BY klant.klantid, klant.voornaam, klant.achternaam. COUNT(reservering.reserveringid) voor NULL-safe count.

✅ Oplossing 
SELECT klant.voornaam, klant.achternaam, 
       COUNT(reservering.reserveringid) AS aantal_reserveringen
FROM klant
LEFT JOIN reservering ON klant.klantid = reservering.klantid
GROUP BY klant.klantid, klant.voornaam, klant.achternaam;

Oefening 4 - Top 5 populairste films

Moeilijk

Vind de top 5 films met de meeste reserveringen. JOIN film + reservering, groepeer, tel, sorteer. Toon filmnaam en aantal_reserveringen.

SQL Query
💡 JOIN + GROUP BY + ORDER BY + LIMIT
💡 Hint

INNER JOIN ON filmid, GROUP BY film.filmid, film.filmnaam, ORDER BY aantal DESC, LIMIT 5

✅ Oplossing 
SELECT film.filmnaam, COUNT(*) AS aantal_reserveringen
FROM film
INNER JOIN reservering ON film.filmid = reservering.filmid
GROUP BY film.filmid, film.filmnaam
ORDER BY aantal_reserveringen DESC
LIMIT 5;

Oefening 5 - Actieve klanten met minimaal 3 reserveringen

Expert

Vind klanten met minimaal 3 reserveringen. Gebruik INNER JOIN + HAVING. Toon voornaam, achternaam en aantal_reserveringen.

SQL Query
💡 JOIN + GROUP BY + HAVING >= 3
💡 Hint

INNER JOIN, GROUP BY klant.klantid + names, HAVING COUNT(*) >= 3. "Minimaal 3" = >= 3.

✅ Oplossing 
SELECT klant.voornaam, klant.achternaam, 
       COUNT(*) AS aantal_reserveringen
FROM klant
INNER JOIN reservering ON klant.klantid = reservering.klantid
GROUP BY klant.klantid, klant.voornaam, klant.achternaam
HAVING COUNT(*) >= 3;

🐛 Veelgemaakte fouten

❌ Ambiguous column name

SELECT filmid, filmnaam
FROM film
INNER JOIN reservering ON filmid = filmid  -- Welke filmid?

Gebruik tabel.kolom voor duidelijkheid!

SELECT film.filmid, film.filmnaam
FROM film
INNER JOIN reservering ON film.filmid = reservering.filmid

❌ Verkeerde JOIN type

-- INNER JOIN: alleen matches (mist klanten zonder reservering)
SELECT COUNT(*)
FROM klant
INNER JOIN reservering ON klant.klantid = reservering.klantid

Gebruik LEFT JOIN voor "alle klanten, ook zonder reservering"!

SELECT COUNT(DISTINCT klant.klantid)
FROM klant
LEFT JOIN reservering ON klant.klantid = reservering.klantid
