04

GROUP BY & Aggregatie

Data groeperen en analyseren met aggregatiefuncties

Voortgang 0/5 oefeningen

💡 Wat ga je leren?

Je hebt geleerd hoe je data uit meerdere tabellen combineert met JOINs. Nu ga je leren hoe je data groepeert en analyseert met aggregatiefuncties!

Aggregatiefuncties

Aggregatiefuncties voeren berekeningen uit over meerdere rijen:

Veelgebruikte aggregatiefuncties 
COUNT(*)        -- Tel aantal rijen
COUNT(kolom)    -- Tel niet-NULL waarden  
SUM(kolom)      -- Som van alle waarden
AVG(kolom)      -- Gemiddelde
MIN(kolom)      -- Kleinste waarde
MAX(kolom)      -- Grootste waarde
Voorbeeld: Tel alle films 
SELECT COUNT(*) AS totaal_films
FROM film;

GROUP BY - Data Groeperen

GROUP BY groepeert rijen met dezelfde waarden. Je kunt dan aggregaties per groep uitvoeren:

Voorbeeld: Aantal films per regisseur 
SELECT regisseur, COUNT(*) AS aantal
FROM film
GROUP BY regisseur;
💡 Belangrijke Regel:

Als je GROUP BY gebruikt, mag je alleen SELECT-en:

  • ✅ Kolommen die in GROUP BY staan
  • ✅ Aggregatiefuncties (COUNT, SUM, AVG, etc.)
  • ❌ NIET: andere kolommen

Query Volgorde

Complete volgorde 
SELECT kolommen, AGG(kolom) AS alias
FROM tabel
WHERE filter_vooraf
GROUP BY kolommen
ORDER BY sorteer
LIMIT aantal;

🎯 Oefeningen

Oefening 1 - Aantal films tellen

Makkelijk

Tel hoeveel films er in totaal in de database zitten. Geef het resultaat de naam totaal_films.

SQL Query
💡 COUNT(*) telt alle rijen
💡 Hint

Gebruik COUNT(*) om alle rijen te tellen, en AS voor de alias.

✅ Oplossing 
SELECT COUNT(*) AS totaal_films
FROM film;

Oefening 2 - Films per regisseur

Gemiddeld

Tel hoeveel films elke regisseur heeft gemaakt. Toon regisseur en aantal_films.

SQL Query
💡 GROUP BY regisseur
💡 Hint

GROUP BY regisseur om per regisseur te groeperen, COUNT(*) om te tellen.

✅ Oplossing 
SELECT regisseur, COUNT(*) AS aantal_films
FROM film
GROUP BY regisseur;

Oefening 3 - Gemiddelde filmduur per genre

Gemiddeld

Bereken de gemiddelde filmduur per genre. Toon genreid en gemiddelde_duur.

SQL Query
💡 AVG voor gemiddelde
💡 Hint

AVG(duur) voor gemiddelde, GROUP BY genreid.

✅ Oplossing 
SELECT genreid, AVG(duur) AS gemiddelde_duur
FROM film
GROUP BY genreid;

Oefening 4 - Top 3 productieve regisseurs

Moeilijk

Vind de 3 regisseurs met de meeste films. Toon regisseur en aantal_films. Sorteer van veel naar weinig.

SQL Query
💡 ORDER BY + DESC + LIMIT
💡 Hint

ORDER BY aantal_films DESC voor meeste eerst, LIMIT 3 voor top 3.

✅ Oplossing 
SELECT regisseur, COUNT(*) AS aantal_films
FROM film
GROUP BY regisseur
ORDER BY aantal_films DESC
LIMIT 3;

Oefening 5 - Totale speeltijd per land

Moeilijk

Bereken de totale speeltijd (SUM van duur) per productieland. Toon landherkomst en totale_speeltijd, gesorteerd van hoog naar laag.

SQL Query
💡 SUM voor totaal
💡 Hint

SUM(duur) voor totaal, GROUP BY landherkomst, ORDER BY totale_speeltijd DESC.

✅ Oplossing 
SELECT landherkomst, SUM(duur) AS totale_speeltijd
FROM film
GROUP BY landherkomst
ORDER BY totale_speeltijd DESC;

🐛 Veelgemaakte fouten

❌ Kolom niet in GROUP BY

SELECT regisseur, filmnaam, COUNT(*)
FROM film
GROUP BY regisseur;  -- filmnaam ontbreekt!

Als je filmnaam wilt tonen, moet je ook op filmnaam groeperen!

SELECT regisseur, COUNT(*)
FROM film
GROUP BY regisseur;

❌ WHERE na GROUP BY

SELECT genreid, COUNT(*)
FROM film
GROUP BY genreid
WHERE COUNT(*) > 5;  -- FOUT!

WHERE komt VOOR GROUP BY. Voor filteren op aggregatie gebruik HAVING (Les 5)!

SELECT genreid, COUNT(*)
FROM film
WHERE landherkomst = 'USA'  -- WHERE eerst
GROUP BY genreid;
