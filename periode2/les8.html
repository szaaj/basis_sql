Je beheerst nu alle belangrijke SQL concepten! In deze les combineer je alles wat je hebt geleerd in complexe queries.

📚 Les 1-3: JOINs & Aggregatie

Basis aggregatie met COUNT, SUM, AVG 
SELECT regisseur, COUNT(*) AS aantal
FROM film
GROUP BY regisseur;

📚 Les 4-5: GROUP BY & HAVING

Filteren op gegroepeerde data 
SELECT genreid, AVG(duur) AS gem
FROM film
GROUP BY genreid
HAVING AVG(duur) > 120;

📚 Les 6: Subqueries

Queries binnen queries 
SELECT filmnaam
FROM film
WHERE duur > (SELECT AVG(duur) FROM film);
💡 Complete Query Volgorde: 
SELECT kolommen, AGG(kolom) AS alias
FROM tabel
WHERE filter_rijen

HAVING filter_groepen

LIMIT aantal;

🎯 Oefeningen - De Ultieme Uitdaging

Oefening 1 - Top 5 langste films

Gemiddeld

Vind de top 5 langste films. Toon filmnaam, regisseur en duur. Sorteer van lang naar kort.

SQL Query
💡 ORDER BY + LIMIT
💡 Hint

ORDER BY duur DESC, LIMIT 5. "Top 5" betekent exact LIMIT 5.

✅ Oplossing 
SELECT filmnaam, regisseur, duur
FROM film
ORDER BY duur DESC
LIMIT 5;

Oefening 2 - Productieve Amerikaanse regisseurs

Moeilijk

Vind Amerikaanse regisseurs met minimaal 3 films. Filter op land (WHERE), groepeer, filter op aantal (HAVING). Toon regisseur en aantal_films.

SQL Query
💡 WHERE + GROUP BY + HAVING
💡 Hint

WHERE landherkomst = 'USA', GROUP BY regisseur, HAVING COUNT(*) >= 3. "Minimaal 3" = >= 3.

✅ Oplossing 
SELECT regisseur, COUNT(*) AS aantal_films
FROM film
WHERE landherkomst = 'USA'
GROUP BY regisseur
HAVING COUNT(*) >= 3;

Oefening 3 - Genres met korte films

Moeilijk

Vind genres met minimaal 2 films die korter zijn dan 100 minuten. Filter op duur (WHERE), groepeer, filter op aantal (HAVING). Toon genreid en aantal_korte_films.

SQL Query
💡 WHERE + GROUP BY + HAVING
💡 Hint

WHERE duur < 100, GROUP BY genreid, HAVING COUNT(*) >= 2. "Korter" = <, "minimaal 2" = >= 2.

✅ Oplossing 
SELECT genreid, COUNT(*) AS aantal_korte_films
FROM film
WHERE duur < 100
GROUP BY genreid
HAVING COUNT(*) >= 2;

Oefening 4 - Regisseurs met lange films boven gemiddelde

Expert

Vind regisseurs met meer dan 1 film die langer zijn dan de gemiddelde filmduur. Gebruik subquery in WHERE. Toon regisseur en aantal_lange_films.

SQL Query
💡 Subquery + GROUP BY + HAVING
💡 Hint

WHERE duur > (SELECT AVG(duur) FROM film), GROUP BY regisseur, HAVING COUNT(*) > 1. "Langer" = >, "meer dan 1" = > 1.

✅ Oplossing 
SELECT regisseur, COUNT(*) AS aantal_lange_films
FROM film
WHERE duur > (SELECT AVG(duur) FROM film)
GROUP BY regisseur
HAVING COUNT(*) > 1;

Oefening 5 - De Ultieme Query

Expert

Vind de top 3 genres met de hoogste gemiddelde filmduur, maar alleen genres met minimaal 5 films. Toon genreid, aantal_films en gemiddelde_duur. Sorteer op gemiddelde (hoog naar laag).

SQL Query
💡 GROUP BY + HAVING + ORDER BY + LIMIT
💡 Hint

GROUP BY genreid, HAVING COUNT(*) >= 5, ORDER BY gemiddelde_duur DESC, LIMIT 3.

✅ Oplossing 
SELECT genreid, COUNT(*) AS aantal_films, AVG(duur) AS gemiddelde_duur
FROM film
GROUP BY genreid
HAVING COUNT(*) >= 5
ORDER BY gemiddelde_duur DESC
LIMIT 3;
