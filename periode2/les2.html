02

GROUP BY - Groeperen

Data groeperen en analyseren

Voortgang 0/8 oefeningen

💡 Wat ga je leren?

In Les 1 heb je geleerd over aggregatiefuncties zoals COUNT() en AVG(). Nu gaan we data groeperen om per categorie te analyseren!

Wat is GROUP BY?

GROUP BY groepeert rijen met dezelfde waarde samen, zodat je per groep kunt tellen, optellen, of gemiddelden berekenen.

Voorbeeld scenario:

Zonder GROUP BY: "Hoeveel films zijn er?" → 1 antwoord

Met GROUP BY: "Hoeveel films per genre?" → Meerdere antwoorden (per genre)

Basis Syntax

Films per regisseur tellen
SELECT regisseur, COUNT(*) 
FROM film 
GROUP BY regisseur;

Groepeert films per regisseur en telt ze per groep

GROUP BY met Meerdere Kolommen

Films per land en jaar
SELECT landherkomst, releasejaar, COUNT(*) 
FROM film 
GROUP BY landherkomst, releasejaar;

Groepeert op combinatie van land EN jaar

GROUP BY met WHERE

Eerst filteren, dan groeperen
SELECT regisseur, COUNT(*) 
FROM film 
WHERE releasejaar >= 2010
GROUP BY regisseur;

Volgorde: WHERE filtert eerst, daarna wordt gegroepeerd

GROUP BY met ORDER BY

Gesorteerde groepen
SELECT regisseur, COUNT(*) AS aantal
FROM film 
GROUP BY regisseur
ORDER BY aantal DESC;

Sorteer de groepen op het aantal (meeste eerst)

⚠️ Belangrijke Regel!

Alle kolommen in SELECT moeten óf:

  • In GROUP BY staan, OF
  • In een aggregatiefunctie zitten (COUNT, SUM, etc.)

❌ Fout: SELECT filmnaam, COUNT(*) GROUP BY regisseur

✅ Goed: SELECT regisseur, COUNT(*) GROUP BY regisseur

💡 Tips:
  • Gebruik AS voor duidelijke namen bij COUNT, SUM, etc.
  • Volgorde: SELECT → FROM → WHERE → GROUP BY → ORDER BY → LIMIT
  • Voor filteren op groepen gebruik je HAVING (komt in Les 3!)

🎯 Oefeningen

Oefening 1 - Films per regisseur

Makkelijk

Tel hoeveel films elke regisseur heeft gemaakt. Toon de regisseur en het aantal films.

SQL Query
💡 GROUP BY regisseur
💡 Hint

SELECT regisseur, COUNT(*) FROM film GROUP BY regisseur

✅ Oplossing 
SELECT regisseur, COUNT(*) AS aantal
FROM film 
GROUP BY regisseur;

Oefening 2 - Films per land

Makkelijk

Hoeveel films komen er uit elk land? Groepeer op landherkomst.

SQL Query
💡 Ctrl+Enter
✅ Oplossing 
SELECT landherkomst, COUNT(*) AS aantal
FROM film 
GROUP BY landherkomst;

Oefening 3 - Gemiddelde duur per genre

Gemiddeld

Wat is de gemiddelde filmduur per genre? Groepeer op genreid en gebruik AVG().

SQL Query
💡 Combineer AVG met GROUP BY
💡 Hint

SELECT genreid, AVG(duur) FROM film GROUP BY genreid

✅ Oplossing 
SELECT genreid, AVG(duur) AS gem_duur
FROM film 
GROUP BY genreid;

Oefening 4 - Films per jaar

Gemiddeld

Hoeveel films zijn er uitgebracht per releasejaar? Sorteer van oud naar nieuw.

SQL Query
💡 GROUP BY + ORDER BY
✅ Oplossing 
SELECT releasejaar, COUNT(*) AS aantal
FROM film 
GROUP BY releasejaar
ORDER BY releasejaar ASC;

Oefening 5 - Top 5 productiefste regisseurs

Gemiddeld

Wie zijn de top 5 regisseurs met de meeste films? Sorteer van meeste naar minste en gebruik LIMIT.

SQL Query
💡 ORDER BY ... DESC LIMIT 5
✅ Oplossing 
SELECT regisseur, COUNT(*) AS aantal
FROM film 
GROUP BY regisseur
ORDER BY aantal DESC
LIMIT 5;

Oefening 6 - Groeperen met WHERE

Moeilijk

Hoeveel films per regisseur zijn er uitgebracht vanaf 2010? Filter eerst met WHERE, groepeer daarna.

SQL Query
💡 WHERE komt vóór GROUP BY
💡 Hint

Volgorde: WHERE releasejaar >= 2010 GROUP BY regisseur

✅ Oplossing 
SELECT regisseur, COUNT(*) AS aantal
FROM film 
WHERE releasejaar >= 2010
GROUP BY regisseur;

Oefening 7 - Meerdere aggregaties

Moeilijk

Toon per land: het aantal films, gemiddelde duur, en langste film. Gebruik duidelijke kolomnamen.

SQL Query
💡 Meerdere functies combineren
✅ Oplossing 
SELECT 
    landherkomst,
    COUNT(*) AS aantal,
    AVG(duur) AS gem_duur,
    MAX(duur) AS langste
FROM film 
GROUP BY landherkomst;

Oefening 8 - Complexe groepering

Moeilijk

Hoeveel films zijn er per land en jaar (combinatie)? Sorteer op land en daarna op jaar.

SQL Query
💡 Groepeer op meerdere kolommen
💡 Hint

GROUP BY landherkomst, releasejaar
ORDER BY landherkomst, releasejaar

✅ Oplossing 
SELECT 
    landherkomst,
    releasejaar,
    COUNT(*) AS aantal
FROM film 
GROUP BY landherkomst, releasejaar
ORDER BY landherkomst, releasejaar;

🐛 Veelgemaakte fouten

❌ Kolom niet in GROUP BY

SELECT filmnaam, regisseur, COUNT(*) 
FROM film 
GROUP BY regisseur;

filmnaam staat niet in GROUP BY en niet in aggregatiefunctie!

SELECT regisseur, COUNT(*) 
FROM film 
GROUP BY regisseur;

❌ GROUP BY na ORDER BY

SELECT regisseur, COUNT(*) 
FROM film 
ORDER BY regisseur
GROUP BY regisseur;

Verkeerde volgorde! GROUP BY komt vóór ORDER BY

SELECT regisseur, COUNT(*) 
FROM film 
GROUP BY regisseur
ORDER BY regisseur;

❌ WHERE met aggregatiefunctie

SELECT regisseur, COUNT(*) 
FROM film 
WHERE COUNT(*) > 2
GROUP BY regisseur;

Voor filteren op COUNT gebruik HAVING (Les 3!)

SELECT regisseur, COUNT(*) 
FROM film 
GROUP BY regisseur
HAVING COUNT(*) > 2;
