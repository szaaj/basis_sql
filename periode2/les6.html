06

HAVING - Filteren op Groepen

Filter geaggregeerde data met HAVING

Voortgang 0/5 oefeningen

💡 Wat ga je leren?

In Les 4 heb je geleerd hoe je data groepeert met GROUP BY. Maar hoe filter je op gegroepeerde resultaten? Daarvoor gebruik je HAVING!

WHERE vs HAVING

Beide filteren data, maar op verschillende momenten:

WHERE - Filter VOOR groeperen 
SELECT regisseur, COUNT(*) AS aantal
FROM film
WHERE landherkomst = 'USA'  -- Filter individuele rijen
GROUP BY regisseur;
HAVING - Filter NA groeperen 
SELECT regisseur, COUNT(*) AS aantal
FROM film
GROUP BY regisseur
HAVING COUNT(*) > 2;  -- Filter gegroepeerde resultaten
💡 Verschil:
  • WHERE: Filtert individuele rijen VOOR GROUP BY
  • HAVING: Filtert groepen NA GROUP BY
  • WHERE: Kan GEEN aggregatiefuncties gebruiken
  • HAVING: Gebruikt aggregatiefuncties (COUNT, SUM, AVG, etc.)

Query Volgorde

Complete volgorde met HAVING 
SELECT kolommen, AGG(kolom) AS alias
FROM tabel
WHERE filter_individuele_rijen
GROUP BY kolommen
HAVING filter_op_aggregaties
ORDER BY sorteer
LIMIT aantal;

Operators in HAVING

Veelgebruikte operators 
HAVING COUNT(*) > 5      -- Meer dan 5
HAVING COUNT(*) >= 5     -- Minimaal 5
HAVING AVG(duur) > 120   -- Gemiddelde hoger dan 120
HAVING SUM(duur) < 500   -- Totaal lager dan 500

🎯 Oefeningen

Oefening 1 - Regisseurs met meer dan 2 films

Gemiddeld

Vind alle regisseurs die meer dan 2 films hebben gemaakt. Toon regisseur en aantal_films.

SQL Query
💡 HAVING COUNT(*) > 2
💡 Hint

"Meer dan 2" betekent > 2 (niet >= 2). Gebruik HAVING voor filteren op COUNT.

✅ Oplossing 
SELECT regisseur, COUNT(*) AS aantal_films
FROM film
GROUP BY regisseur
HAVING COUNT(*) > 2;

Oefening 2 - Genres met lange films

Gemiddeld

Vind genres waar de gemiddelde filmduur hoger is dan 120 minuten. Toon genreid en gemiddelde_duur.

SQL Query
💡 HAVING AVG(duur) > 120
💡 Hint

"Hoger dan 120" betekent > 120 (niet >= 120). Gebruik AVG(duur) in HAVING.

✅ Oplossing 
SELECT genreid, AVG(duur) AS gemiddelde_duur
FROM film
GROUP BY genreid
HAVING AVG(duur) > 120;

Oefening 3 - Amerikaanse regisseurs met minimaal 3 films

Moeilijk

Vind Amerikaanse regisseurs met minimaal 3 films. Filter eerst op land (WHERE), dan op aantal (HAVING). Toon regisseur en aantal_films.

SQL Query
💡 WHERE eerst, dan HAVING
💡 Hint

WHERE landherkomst = 'USA' VOOR GROUP BY. "Minimaal 3" betekent >= 3 in HAVING.

✅ Oplossing 
SELECT regisseur, COUNT(*) AS aantal_films
FROM film
WHERE landherkomst = 'USA'
GROUP BY regisseur
HAVING COUNT(*) >= 3;

Oefening 4 - Top 3 populairste genres

Moeilijk

Vind de top 3 genres met de meeste films, maar alleen genres met minimaal 5 films. Toon genreid en aantal_films. Sorteer van veel naar weinig.

SQL Query
💡 HAVING + ORDER BY + LIMIT
💡 Hint

"Minimaal 5" = >= 5, "top 3" = LIMIT 3, "meeste films" = ORDER BY aantal_films DESC

✅ Oplossing 
SELECT genreid, COUNT(*) AS aantal_films
FROM film
GROUP BY genreid
HAVING COUNT(*) >= 5
ORDER BY aantal_films DESC
LIMIT 3;

Oefening 5 - Landen met korte films

Moeilijk

Vind landen met minimaal 2 films die korter zijn dan 100 minuten. Filter eerst op duur (WHERE), tel per land, filter dan op aantal (HAVING). Toon landherkomst en aantal_korte_films.

SQL Query
💡 WHERE + HAVING combineren
💡 Hint

"Korter dan 100" = < 100 in WHERE. "Minimaal 2" = >= 2 in HAVING.

✅ Oplossing 
SELECT landherkomst, COUNT(*) AS aantal_korte_films
FROM film
WHERE duur < 100
GROUP BY landherkomst
HAVING COUNT(*) >= 2;

🐛 Veelgemaakte fouten

❌ WHERE op aggregatie

SELECT regisseur, COUNT(*)
FROM film
WHERE COUNT(*) > 2       -- FOUT!
GROUP BY regisseur;

WHERE kan niet filteren op COUNT/SUM/AVG. Gebruik HAVING!

SELECT regisseur, COUNT(*)
FROM film
GROUP BY regisseur
HAVING COUNT(*) > 2;

❌ HAVING voor GROUP BY

SELECT genreid, COUNT(*)
FROM film
HAVING COUNT(*) > 5      -- Te vroeg!
GROUP BY genreid;

HAVING komt ALTIJD NA GROUP BY!

SELECT genreid, COUNT(*)
FROM film
GROUP BY genreid
HAVING COUNT(*) > 5;
