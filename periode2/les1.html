Aggregatiefuncties

COUNT, SUM, AVG, MIN, MAX

Voortgang 0/7 oefeningen

💡 Wat ga je leren?

In Periode 1 heb je geleerd hoe je data ophaalt en filtert. Nu gaan we data analyseren met aggregatiefuncties!

Wat zijn Aggregatiefuncties?

Functies die meerdere rijen samen vatten tot één resultaat. Bijvoorbeeld: "Hoeveel films hebben we?" of "Wat is de gemiddelde duur?"

COUNT() - Tellen

Aantal films tellen
SELECT COUNT(*) FROM film;

Telt ALLE rijen in de tabel

Unieke genres tellen
SELECT COUNT(DISTINCT genreid) FROM film;

Telt alleen unieke waarden (geen duplicaten)

SUM() - Optellen

Totale filmduur
SELECT SUM(duur) FROM film;

Telt alle duuren bij elkaar op

AVG() - Gemiddelde

Gemiddelde filmduur
SELECT AVG(duur) FROM film;

Berekent het gemiddelde van alle duuren

MIN() & MAX() - Kleinste en Grootste

Kortste en langste film
SELECT MIN(duur), MAX(duur) FROM film;

Vindt de kleinste en grootste waarde

💡 Tips:
  • Aggregatiefuncties geven 1 rij terug (tenzij je GROUP BY gebruikt)
  • Je kunt ze combineren: SELECT COUNT(*), AVG(duur) FROM film
  • Gebruik AS voor duidelijke kolomnamen
  • NULL waarden worden genegeerd (behalve bij COUNT(*))

Makkelijk

SQL Query
💡 Ctrl+Enter

SQL Query
💡 Ctrl+Enter

Oefening 5 - Duidelijke kolomnamen (AS)

Gemiddeld

Tel het aantal films en geef de kolom de naam totaal_films (gebruik AS).

SQL Query
💡 Gebruik AS voor alias
💡 Hint

SELECT COUNT(*) AS totaal_films FROM...

✅ Oplossing 
SELECT COUNT(*) AS totaal_films FROM film;

Oefening 6 - Meerdere statistieken

Gemiddeld

Haal in één query op: aantal films, gemiddelde duur, kortste duur, en langste duur. Geef elke kolom een duidelijke naam.

SQL Query
💡 Combineer functies met komma's
✅ Oplossing 
SELECT 
    COUNT(*) AS aantal,
    AVG(duur) AS gemiddelde,
    MIN(duur) AS kortste,
    MAX(duur) AS langste
FROM film;

Oefening 7 - COUNT met WHERE

Moeilijk

Tel hoeveel films er zijn die langer dan 120 minuten duren.

SQL Query
💡 Combineer COUNT met WHERE
💡 Hint

SELECT COUNT(*) FROM film WHERE duur > 120

✅ Oplossing 
SELECT COUNT(*) AS lange_films 
FROM film 
WHERE duur > 120;

🐛 Veelgemaakte fouten

❌ COUNT zonder haakjes

SELECT COUNT FROM film;

COUNT is een functie, gebruik COUNT(*) of COUNT(kolom)

SELECT COUNT(*) FROM film;

❌ Aggregatie met normale kolommen

SELECT filmnaam, COUNT(*) FROM film;

Je kunt niet een specifieke film EN een totaal tellen tegelijk!

SELECT COUNT(*) FROM film;

❌ WHERE na GROUP BY (komt in Les 2)

SELECT COUNT(*) FROM film GROUP BY genreid WHERE COUNT(*) > 5;

Voor filteren op aggregatie gebruik je HAVING, niet WHERE

SELECT COUNT(*) FROM film GROUP BY genreid HAVING COUNT(*) > 5;
