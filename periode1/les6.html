06

Data Wijzigen en Verwijderen

UPDATE en DELETE - ⚠️ PAS OP!

Voortgang 0/8 oefeningen

💡 Wat ga je leren?

Vandaag leer je hoe je bestaande data wijzigt (UPDATE) en verwijdert (DELETE).

⚠️ GEVAAR! Heel belangrijk!

ALTIJD een WHERE gebruiken bij UPDATE en DELETE!

Zonder WHERE verander/verwijder je ALLE rijen in de tabel! 😱

UPDATE - Data wijzigen

Basis syntax 
UPDATE tabel 
SET kolom1 = 'nieuwe waarde', kolom2 = 'andere waarde'
WHERE voorwaarde;
Voorbeeld: Telefoonnummer aanpassen 
UPDATE klant 
SET telefoon = '0612345678' 
WHERE idklant = 5;

DELETE - Data verwijderen

Basis syntax 
DELETE FROM tabel 
WHERE voorwaarde;
Voorbeeld: Klant verwijderen 
DELETE FROM klant 
WHERE idklant = 5;
💡 Veilig werken:
  • Test EERST met SELECT om te zien welke rijen je raakt
  • Gebruik ALTIJD WHERE met een uniek ID
  • In productie: maak eerst een backup!
  • Twijfel je? Vraag eerst om hulp!
  • ⚠️ Database reset: Refresh de pagina om wijzigingen ongedaan te maken!
  • ℹ️ FOREIGN KEY: Je kan data met relaties niet zomaar verwijderen. Meer hierover in Periode 2!

🎯 Oefeningen

Oefening 1 - Telefoon nummer wijzigen

Makkelijk

Update het telefoonnummer van klant met ID 1 naar 0698765432.

SQL Query
⚠️ Vergeet WHERE niet!
💡 Hint

UPDATE klant SET telefoon = '...' WHERE idklant = 1

✅ Oplossing 
UPDATE klant 
SET telefoon = '0698765432' 
WHERE idklant = 1;

Oefening 2 - Prijs aanpassen

Makkelijk

Verlaag de prijs van reis 1 naar 799.00 euro.

SQL Query
⚠️ WHERE niet vergeten!
✅ Oplossing 
UPDATE reis 
SET prijsperpersoon = 799.00 
WHERE idreis = 1;

Oefening 3 - Meerdere kolommen wijzigen

Gemiddeld

Update klant 2: naam naar Bakker-Jansen A. en email naar bakker.jansen@email.nl.

SQL Query
💡 Gebruik komma tussen SET kolommen
💡 Hint

SET kolom1 = waarde1, kolom2 = waarde2 WHERE ...

✅ Oplossing 
UPDATE klant 
SET naam = 'Bakker-Jansen A.', email = 'bakker.jansen@email.nl' 
WHERE idklant = 2;

Oefening 4 - Veiligheidscheck met SELECT

Gemiddeld

Voer EERST een SELECT uit om te zien welke klanten in Amsterdam wonen, voordat je ze zou updaten.

SQL Query
💡 Dit is een SELECT, geen UPDATE!
💡 Hint

Dit is gewoon een normale SELECT WHERE query. Zo check je VOOR je UPDATE doet!

✅ Oplossing 
SELECT * FROM klant 
WHERE plaats = 'Amsterdam';

Oefening 5 - DELETE: Klant verwijderen

Gemiddeld

Verwijder klant met ID 3.

SQL Query
⚠️ GEVAAR: WHERE IS VERPLICHT!
✅ Oplossing 
DELETE FROM klant 
WHERE idklant = 3;

Oefening 6 - DELETE: Reis verwijderen

Gemiddeld

Verwijder reis met ID 2.

SQL Query
⚠️ WHERE IS VERPLICHT!
✅ Oplossing 
DELETE FROM reis 
WHERE idreis = 2;

Oefening 7 - DELETE met voorwaarde

Moeilijk

Verwijder ALLE reizen die meer dan 3000 euro kosten.

SQL Query
💡 Test eerst met SELECT!
💡 Hint

DELETE FROM reis WHERE prijsperpersoon > 3000

✅ Oplossing 
DELETE FROM reis 
WHERE prijsperpersoon > 3000;

Oefening 8 - UPDATE met berekening (Bonus)

Bonus

Verhoog de prijs van reis 1 met 10%. Gebruik een berekening!

SQL Query
💡 Je kunt rekenen: SET prijs = prijs * 1.10
💡 Hint

SET prijsperpersoon = prijsperpersoon * 1.10 (= +10%)

✅ Oplossing 
UPDATE reis 
SET prijsperpersoon = prijsperpersoon * 1.10 
WHERE idreis = 1;

🐛 Veelgemaakte fouten

❌❌❌ WHERE VERGETEN - GEVAARLIJK!

UPDATE klant SET plaats = 'Amsterdam';

DIT VERANDERT ALLE KLANTEN! Altijd WHERE gebruiken!

UPDATE klant SET plaats = 'Amsterdam' WHERE idklant = 5;

❌ SET keyword vergeten

UPDATE klant naam = 'Jansen' WHERE id = 1;

Vergeet SET niet!

UPDATE klant SET naam = 'Jansen' WHERE id = 1;

❌ Komma's verkeerd bij meerdere kolommen

UPDATE klant SET naam = 'X' AND plaats = 'Y' WHERE id = 1;

Gebruik komma, niet AND!

UPDATE klant SET naam = 'X', plaats = 'Y' WHERE id = 1;

❌ DELETE zonder WHERE - gekoppelde data

DELETE FROM bestemming WHERE id = 5;

Als er reizen gekoppeld zijn, krijg je een foreign key error!

-- Eerst reizen verwijderen, dan bestemming
DELETE FROM reis WHERE idbestemming = 5;
DELETE FROM bestemming WHERE idbestemming = 5;
