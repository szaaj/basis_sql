Leer SQL sneller met AI als je persoonlijke assistent

💡 Wat is de AI SQL Coach?

De AI SQL Coach helpt je om beter en sneller SQL te leren! Je kunt AI tools zoals Copilot gebruiken om:

  • ✅ Moeilijke concepten uit te laten leggen
  • ✅ Je errors te debuggen
  • ✅ Extra voorbeelden te krijgen
  • ✅ Je begrip te controleren

🎯 De Gouden Regel

Probeer ALTIJD eerst zelf (minimaal 5 minuten)!

Gebruik AI pas als je echt vast zit. Zo leer je het beste en onthoud je het langste! 🧠

✅ WEL Doen vs ❌ NIET Doen

✅ WEL

  • Vraag om uitleg
  • Vraag om voorbeelden
  • Laat je denken checken
  • Vraag "waarom" het werkt
  • Gebruik voor debugging

❌ NIET

  • Directe antwoorden vragen
  • Copy-pasten zonder begrijpen
  • AI alles laten invullen
  • Niet eerst zelf proberen
  • Valsspelen bij opdrachten

📱 Welke AI Tool Gebruik Je?

Kies een AI tool die je wilt gebruiken. Ze werken allemaal goed voor SQL leren!

🤖 Copilot (Aanbevolen) 💬 ChatGPT 🧠 Claude

📝 Copy-Paste Prompts - Gewoon Kopiëren!

Klik op "Kopiëren", plak in je AI tool, en pas aan naar jouw situatie! 👇

💬 UITLEG VRAGEN

Ik leer SQL en snap het concept van SELECT nog niet helemaal. Kun je uitleggen wat SELECT doet, wanneer ik het gebruik, en een simpel voorbeeld geven?

💬 UITLEG VRAGEN

Wat is het verschil tussen < en <= in SQL? Leg het uit met een voorbeeld zodat ik het snap!

🐛 DEBUGGING

Ik krijg deze error: "no such column: naam". Wat betekent dit en hoe los ik het op? (Geef geen directe oplossing maar leg uit waar ik op moet letten!)

🐛 DEBUGGING

Mijn query geeft duplicate resultaten. Welke SQL trucjes kan ik gebruiken om duplicates weg te halen? Leg uit hoe ze werken!

🤔 DENKEN CHECKEN

Ik moet alle reizen onder 1000 euro tonen, gesorteerd op prijs. Mijn aanpak is: [beschrijf wat jij denkt]. Klopt dit? Geef hints maar geen complete oplossing!

🤔 DENKEN CHECKEN

Ik heb deze query: [plak je query]. Kun je stap voor stap uitleggen WAT mijn query doet? Dan zie ik of ik het goed snap!

🎓 MEER OEFENEN

Ik wil ORDER BY beter oefenen. Kun je 3 oefeningen bedenken (van makkelijk naar moeilijk)? Geef GEEN oplossingen, alleen de opdrachten!

🎓 MEER OEFENEN

Waarom moet ik quotes gebruiken bij tekst ('Amsterdam') maar niet bij getallen? Leg het verschil uit tussen TEXT en INTEGER!

🎮 Hoe Werkt Het?

Stap 1: Klik op "Kopiëren" bij een prompt hierboven 👆

Stap 2: Open Copilot, ChatGPT of Claude 🤖

Stap 3: Plak de prompt (Ctrl+V) 📋

Stap 4: Pas aan naar jouw situatie ✏️

Stap 5: Lees de uitleg en probeer het zelf! 💪
