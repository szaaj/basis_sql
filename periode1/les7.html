🎯 Mixed Oefeningen

Deze oefeningen combineren alles wat je geleerd hebt. Geen hints - je kunt het!

Oefening 1 - Top 5 goedkoopste West-Europese reizen

Gemiddeld

Haal de 5 goedkoopste reizen op naar bestemmingen in West-Europa. Toon alleen plaats, land, prijsperpersoon en aantaldagen.

SQL Query
💡 Combineer alles: specifieke kolommen, WHERE, ORDER BY, LIMIT
✅ Oplossing 
-- Vereenvoudigde versie (zonder JOIN):
SELECT plaats, land, werelddeel FROM bestemming 
WHERE werelddeel = 'West-Europa' 
LIMIT 5;

-- Met JOIN (Periode 2):
SELECT b.plaats, b.land, r.prijsperpersoon, r.aantaldagen 
FROM reis r
JOIN bestemming b ON r.idbestemming = b.idbestemming
WHERE b.werelddeel = 'West-Europa'
ORDER BY r.prijsperpersoon ASC
LIMIT 5;

Oefening 2 - Klanten uit grote steden

Gemiddeld

Haal alle klanten op uit Amsterdam, Rotterdam, Utrecht OF Den Haag, gesorteerd op plaats en dan op naam.

SQL Query
💡 Gebruik OR + ORDER BY met meerdere kolommen
✅ Oplossing 
SELECT * FROM klant 
WHERE plaats = 'Amsterdam' 
   OR plaats = 'Rotterdam' 
   OR plaats = 'Utrecht' 
   OR plaats = 'Den Haag'
ORDER BY plaats ASC, naam ASC;

Oefening 3 - Complete CRUD operatie

Moeilijk

1. Voeg een nieuwe klant toe: ID 200, naam "Test Gebruiker", plaats "Teststad"
2. Update de plaats naar "Amsterdam"
3. Verwijder de klant weer

SQL Query
💡 3 statements: INSERT, UPDATE, DELETE
✅ Oplossing 
INSERT INTO klant (idklant, naam, plaats)
VALUES (200, 'Test Gebruiker', 'Teststad');

UPDATE klant 
SET plaats = 'Amsterdam' 
WHERE idklant = 200;

DELETE FROM klant 
WHERE idklant = 200;

Oefening 4 - Betaalbare weekreizen

Moeilijk

Haal reizen op die precies 7 dagen duren EN tussen 500 en 1500 euro kosten, gesorteerd van goedkoop naar duur.

SQL Query
💡 WHERE met AND, gebruik BETWEEN of >= en <=
✅ Oplossing 
SELECT * FROM reis 
WHERE aantaldagen = 7 
  AND prijsperpersoon >= 500 
  AND prijsperpersoon <= 1500
ORDER BY prijsperpersoon ASC;

-- Of met BETWEEN:
SELECT * FROM reis 
WHERE aantaldagen = 7 
  AND prijsperpersoon BETWEEN 500 AND 1500
ORDER BY prijsperpersoon ASC;

Oefening 5 - Data analyse

Moeilijk

Haal alle bestemmingen BUITEN Europa op die nog niet verwijderd zijn, toon alleen plaats en werelddeel, gesorteerd alfabetisch op plaats.

SQL Query
💡 NOT of <>, specifieke kolommen, ORDER BY
✅ Oplossing 
SELECT plaats, werelddeel FROM bestemming 
WHERE werelddeel <> 'Europa'
ORDER BY plaats ASC;

-- Of met NOT:
SELECT plaats, werelddeel FROM bestemming 
WHERE NOT werelddeel = 'West-Europa'
ORDER BY plaats ASC;
