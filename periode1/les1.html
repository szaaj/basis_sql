Introductie + Eerste SELECT queries

Leer de basis: tabellen bekijken en data ophalen

Voortgang 0/6 oefeningen

💡 Wat ga je leren?

Vandaag beginnen we met SQL! Je leert hoe je data uit een database kunt ophalen met SELECT queries.

📚 De Reizen Database

We werken met een database van een reisbureau. De database heeft 4 tabellen:

  • klant - Klantgegevens (naam, adres, email, etc.)
  • reis - Reizen (bestemming, datum, prijs, duur)
  • bestemming - Bestemmingen (plaats, land, werelddeel)
  • boeking - Boekingen van klanten voor reizen

SELECT - Data ophalen

Met SELECT haal je data op uit een tabel. Dit is de meest gebruikte SQL query!

Voorbeeld: Alle data uit een tabel 
SELECT * FROM klant;

Dit haalt alle kolommen van alle rijen op uit de tabel "klant".

Voorbeeld: Specifieke kolommen 
SELECT naam, plaats FROM klant;

Dit haalt alleen de kolommen "naam" en "plaats" op.

DISTINCT - Unieke waarden

Soms wil je alleen unieke (verschillende) waarden zien, zonder duplicaten.

Voorbeeld: Welke verschillende plaatsen hebben we? 
SELECT DISTINCT plaats FROM klant;

Dit toont elke plaats maar één keer, ook al wonen er meerdere klanten in dezelfde plaats.

💡 Tip: Gebruik * om alle kolommen op te halen, maar in de praktijk selecteer je meestal specifieke kolommen die je nodig hebt. Gebruik DISTINCT om duplicaten te verwijderen.

🐛 Veelgemaakte fouten

❌ Fout: Tabel naam verkeerd gespeld

SELECT * FROM klanten;

De tabel heet klant, niet klanten!

SELECT * FROM klant;

❌ Fout: Kolom naam verkeerd gespeld

SELECT naam, woonplaats FROM klant;

De kolom heet plaats, niet woonplaats.

SELECT naam, plaats FROM klant;

❌ Fout: Puntkomma vergeten

SELECT * FROM klant

Altijd een puntkomma ; aan het einde!

SELECT * FROM klant;
