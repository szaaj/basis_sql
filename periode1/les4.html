04

Sorteren en Beperken

ORDER BY en LIMIT

Voortgang 0/8 oefeningen

💡 Wat ga je leren?

Vandaag leer je hoe je resultaten sorteert (ORDER BY) en hoe je het aantal resultaten beperkt (LIMIT).

ORDER BY - Sorteren

Oplopend sorteren (ASC = Ascending) 
SELECT * FROM reis 
ORDER BY prijsperpersoon ASC;

Goedkoopste reizen eerst

Aflopend sorteren (DESC = Descending) 
SELECT * FROM reis 
ORDER BY prijsperpersoon DESC;

Duurste reizen eerst

LIMIT - Aantal resultaten beperken

Top 5 goedkoopste reizen 
SELECT * FROM reis 
ORDER BY prijsperpersoon ASC 
LIMIT 5;

Combineren met WHERE

Top 3 duurste reizen in Europa 
SELECT * FROM reis 
WHERE werelddeel = 'Europa' 
ORDER BY prijsperpersoon DESC 
LIMIT 3;
💡 Volgorde is belangrijk!
  • 1. SELECT
  • 2. FROM
  • 3. WHERE
  • 4. ORDER BY
  • 5. LIMIT

🎯 Oefeningen

Oefening 1 - Alfabetisch sorteren

Makkelijk

Haal alle klanten op, gesorteerd op naam (alfabetisch, A-Z).

SQL Query
💡 Ctrl+Enter
💡 Hint

ORDER BY naam ASC (of gewoon ORDER BY naam, ASC is standaard)

✅ Oplossing 
SELECT * FROM klant 
ORDER BY naam ASC;

Oefening 2 - Duurste reizen eerst

Makkelijk

Haal alle reizen op, gesorteerd van duur naar goedkoop.

SQL Query
💡 Ctrl+Enter
✅ Oplossing 
SELECT * FROM reis 
ORDER BY prijsperpersoon DESC;

Oefening 3 - Top 5 goedkoopste

Makkelijk

Haal de 5 goedkoopste reizen op.

SQL Query
💡 Ctrl+Enter
💡 Hint

ORDER BY prijsperpersoon ASC, dan LIMIT 5

✅ Oplossing 
SELECT * FROM reis 
ORDER BY prijsperpersoon ASC 
LIMIT 5;

Oefening 4 - Top 3 langste reizen

Gemiddeld

Haal de 3 langste reizen op (meeste dagen).

SQL Query
💡 Ctrl+Enter
✅ Oplossing 
SELECT * FROM reis 
ORDER BY aantaldagen DESC 
LIMIT 3;

Oefening 5 - Eerste 10 klanten

Gemiddeld

Haal de eerste 10 klanten op, gesorteerd op ID.

SQL Query
💡 Ctrl+Enter
✅ Oplossing 
SELECT * FROM klant 
ORDER BY idklant ASC 
LIMIT 10;

Oefening 6 - Goedkope weekreizen

Gemiddeld

Haal de 5 goedkoopste reizen van 7 dagen op.

SQL Query
💡 Ctrl+Enter
💡 Hint

Combineer WHERE (7 dagen) met ORDER BY en LIMIT

✅ Oplossing 
SELECT * FROM reis 
WHERE aantaldagen = 7 
ORDER BY prijsperpersoon ASC 
LIMIT 5;

Oefening 7 - Duurste Europese reizen

Moeilijk

Haal de 3 duurste Europese bestemmingen op. Toon alleen plaats, land en geef de reis-prijs weer.

SQL Query
💡 Ctrl+Enter
💡 Hint

Dit vereist eigenlijk JOIN (periode 2). Voor nu: selecteer specifieke kolommen uit bestemming tabel

✅ Oplossing (vereenvoudigd) 
SELECT plaats, land, werelddeel FROM bestemming 
WHERE werelddeel = 'Europa' 
ORDER BY idbestemming DESC 
LIMIT 3;

Oefening 8 - Meerdere sorteer criteria

Bonus

Haal alle reizen op, eerst gesorteerd op aantal dagen (laag-hoog), en binnen dezelfde duur op prijs (laag-hoog).

SQL Query
💡 Ctrl+Enter
💡 Hint

Je kunt meerdere kolommen opgeven: ORDER BY kolom1, kolom2

✅ Oplossing 
SELECT * FROM reis 
ORDER BY aantaldagen ASC, prijsperpersoon ASC;

🐛 Veelgemaakte fouten

❌ Verkeerde volgorde keywords

SELECT * FROM reis LIMIT 5 ORDER BY prijs

ORDER BY moet voor LIMIT!

SELECT * FROM reis ORDER BY prijs LIMIT 5

❌ WHERE na ORDER BY

SELECT * FROM reis ORDER BY prijs WHERE dagen = 7

WHERE moet voor ORDER BY!

SELECT * FROM reis WHERE dagen = 7 ORDER BY prijs

❌ Verkeerde kolom voor ORDER BY

SELECT naam FROM klant ORDER BY prijs

Klant tabel heeft geen prijs kolom!

SELECT naam FROM klant ORDER BY naam
