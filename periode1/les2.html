02

Filteren met WHERE

Specifieke rijen selecteren met voorwaarden

Voortgang 0/7 oefeningen

💡 Wat ga je leren?

In les 1 heb je geleerd hoe je alle data uit een tabel kunt ophalen. Nu leer je hoe je specifieke rijen selecteert met de WHERE clausule.

WHERE - Rijen filteren

Met WHERE voeg je voorwaarden toe aan je query. Alleen rijen die aan de voorwaarde voldoen worden getoond.

Voorbeeld: Klanten uit Amsterdam 
SELECT * FROM klant WHERE plaats = 'Amsterdam';

Dit haalt alleen klanten op die in Amsterdam wonen.

Vergelijkingsoperatoren

Je kunt verschillende operatoren gebruiken om te vergelijken:

= Is gelijk aan WHERE prijs = 500
<> of != Niet gelijk aan WHERE plaats <> 'Utrecht'
> Groter dan WHERE dagen > 7
< Kleiner dan WHERE prijs < 1000
>= Groter of gelijk WHERE dagen >= 14
<= Kleiner of gelijk WHERE prijs <= 2000
LIKE Patroon zoeken WHERE naam LIKE 'Jan%'

LIKE - Zoeken op patronen

Met LIKE kun je zoeken op gedeeltelijke overeenkomsten. Gebruik % als wildcard (= willekeurige tekens).

Begint met "Jan" 
SELECT * FROM klant WHERE naam LIKE 'Jan%';

Vindt: "Jansen", "Jansma", "Jan de Vries"

Eindigt op "dam" 
SELECT * FROM klant WHERE plaats LIKE '%dam';

Vindt: "Amsterdam", "Rotterdam"

Bevat "dam" ergens 
SELECT * FROM klant WHERE plaats LIKE '%dam%';

Vindt: "Amsterdam", "Rotterdam", "Volendam", "Zaandam"

💡 Belangrijke regels:
  • Tekst moet altijd tussen enkele quotes: 'Amsterdam'
  • Getallen zonder quotes: 14
  • Let op hoofdletters: 'rotterdam' ≠ 'Rotterdam'

Oefening 1 - Klanten uit Rotterdam

Makkelijk

Haal alle klanten op die in Rotterdam wonen.

SQL Query
💡 Ctrl+Enter om uit te voeren
💡 Hint

Gebruik WHERE met de kolom plaats en vergeet de quotes niet bij tekst!

✅ Oplossing

SQL Query

💡 Hint

🐛 Veelgemaakte fouten

❌ Quotes vergeten

WHERE plaats = Rotterdam

Tekst MOET tussen quotes!

WHERE plaats = 'Rotterdam'

❌ Hoofdletters niet correct

WHERE plaats = 'rotterdam'

Als data 'Rotterdam' is, vindt dit niets.

WHERE plaats = 'Rotterdam'

❌ NULL vergelijken

WHERE email = NULL

Voor NULL gebruik speciale syntax:

WHERE email IS NULL
