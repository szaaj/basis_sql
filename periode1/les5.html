05

Data Toevoegen met INSERT

Nieuwe rijen aan een tabel toevoegen

Voortgang 0/6 oefeningen

💡 Wat ga je leren?

Tot nu toe heb je alleen data opgehaald. Nu leer je hoe je nieuwe data toevoegt aan een tabel!

INSERT - Nieuwe rij toevoegen

Basis syntax 
INSERT INTO tabel (kolom1, kolom2, kolom3)
VALUES ('waarde1', 'waarde2', 'waarde3');
Voorbeeld: Nieuwe klant toevoegen 
INSERT INTO klant (naam, plaats, telefoon)
VALUES ('Jansen P.', 'Amsterdam', '0612345678');

✅ AUTO_INCREMENT

✅ Deze database gebruikt AUTO_INCREMENT!

Je hoeft geen ID op te geven - de database doet dat automatisch!

-- ID wordt automatisch toegekend
INSERT INTO klant (naam, plaats, telefoon)
VALUES ('Bakker A.', 'Rotterdam', '0612345678');

💡 Elke nieuwe INSERT krijgt automatisch het volgende beschikbare ID

⚠️ Let op: Refresh de pagina om de database te resetten naar de beginstand!

Meerdere rijen tegelijk

Meerdere INSERT in één keer 
INSERT INTO klant (naam, plaats)
VALUES 
    ('Bakker A.', 'Rotterdam'),
    ('Visser B.', 'Utrecht'),
    ('Smit C.', 'Den Haag');
💡 Tips:
  • Geen ID opgeven - AUTO_INCREMENT regelt dit!
  • Tekst tussen enkele quotes: 'Amsterdam'
  • Getallen zonder quotes: 1500
  • Datums tussen quotes: '2024-12-25'
  • Decimalen met punt: 199.50

🎯 Oefeningen

Oefening 1 - Eerste klant toevoegen

Makkelijk

Voeg een nieuwe klant toe met naam Jansen P., plaats Amsterdam. Het ID wordt automatisch toegekend!

SQL Query
💡 Ctrl+Enter
💡 Hint

INSERT INTO klant (idklant, naam, plaats) VALUES (...);

✅ Oplossing 
INSERT INTO klant (naam, plaats)
VALUES ('Jansen P.', 'Amsterdam');

Oefening 2 - Klant met alle gegevens

Makkelijk

Voeg een klant toe met naam Bakker A., plaats Rotterdam, telefoon 0687654321, email bakker@email.nl. Het ID wordt automatisch toegekend!

SQL Query
💡 Ctrl+Enter
✅ Oplossing 
INSERT INTO klant (naam, plaats, telefoon, email)
VALUES ('Bakker A.', 'Rotterdam', '0687654321', 'bakker@email.nl');

Oefening 3 - Nieuwe bestemming

Gemiddeld

Voeg een nieuwe bestemming toe: ID 'STOCK' (TEXT!), plaats Stockholm, land Zweden, werelddeel Europa. Let op: bestemming IDs zijn TEXT, geen nummers!

SQL Query
💡 Ctrl+Enter
✅ Oplossing 
INSERT INTO bestemming (idbestemming, plaats, land, werelddeel)
VALUES ('STOCK', 'Stockholm', 'Zweden', 'Europa');

Oefening 4 - Nieuwe reis

Gemiddeld

Voeg een reis toe naar bestemming 'STOCK' (Stockholm), vertrekdatum 2024-06-15, dagen 5, prijs 899.50. Het reis-ID wordt automatisch toegekend!

SQL Query
💡 Ctrl+Enter
💡 Hint

Let op: datums tussen quotes, getallen zonder quotes, decimalen met punt!

✅ Oplossing 
INSERT INTO reis (idbestemming, vertrekdatum, aantaldagen, prijsperpersoon)
VALUES ('STOCK', '2024-06-15', 5, 899.50);

Oefening 6 - Complete reis ervaring (Bonus)

Moeilijk

Voeg een complete reis toe met bestemming: ID 'LISBN' (TEXT!): Lissabon, Portugal, Europa
En reis naar 'LISBN', datum 2024-08-01, 10 dagen, 1299.00 euro (reis-ID automatisch!)

SQL Query
💡 Je moet 2 INSERT statements schrijven!
💡 Hint

Eerst bestemming toevoegen, dan reis. Twee INSERT statements na elkaar (beide met puntkomma).

✅ Oplossing 
INSERT INTO bestemming (idbestemming, plaats, land, werelddeel)
VALUES ('LISBN', 'Lissabon', 'Portugal', 'Europa');

INSERT INTO reis (idbestemming, vertrekdatum, aantaldagen, prijsperpersoon)
VALUES ('LISBN', '2024-08-01', 10, 1299.00);

🐛 Veelgemaakte fouten

❌ Quotes vergeten bij tekst

INSERT INTO klant (naam) VALUES (Jansen);

Tekst moet tussen quotes!

INSERT INTO klant (naam) VALUES ('Jansen');

❌ Aantal kolommen ≠ aantal waarden

INSERT INTO klant (naam, plaats) VALUES ('Jansen');

2 kolommen maar 1 waarde!

INSERT INTO klant (naam, plaats) VALUES ('Jansen', 'Amsterdam');

❌ ID al bestaat (duplicate key)

INSERT INTO klant (idklant, naam) VALUES (1, 'Test');

ID 1 bestaat al! Kies een nieuw ID.

SELECT MAX(idklant) FROM klant;  -- Kijk wat hoogste ID is
INSERT INTO klant (idklant, naam) VALUES (100, 'Test');

❌ Datum verkeerd formaat

INSERT INTO reis (vertrekdatum) VALUES ('15-06-2024');

Gebruik YYYY-MM-DD formaat!

INSERT INTO reis (vertrekdatum) VALUES ('2024-06-15');

❌ Komma in decimaal getal

INSERT INTO reis (prijs) VALUES (899,50);

Gebruik PUNT, geen komma!

INSERT INTO reis (prijs) VALUES (899.50);
